# 🤖 MDH BOT - WhatsApp Multi-Device Bot
![Version](https://img.shields.io/badge/Version-3.0.7-blue.svg) ![Node.js](https://img.shields.io/badge/Node.js-Ready-green.svg) ![WhatsApp](https://img.shields.io/badge/WhatsApp-Web.js-25D366.svg)

**MDH BOT** is a powerful, customizable, and secure WhatsApp Bot built using Node.js and `whatsapp-web.js`. It features a built-in web dashboard for easy 8-digit Pairing Code linking (bypassing QR code scans) and includes Anti-Crash technology.

Created by **MRDIEHARD** (samkelo Mdh).

🔗 **[Subscribe on YouTube](https://youtube.com/@samkeloMdh)**

---

## ✨ Features
* 🌐 **Web Dashboard Pairing:** Link your bot using an 8-character pairing code directly from a custom UI.
* 🛡️ **Anti-Crash Tech:** Background headless browser optimized to prevent crashes and memory leaks.
* 💾 **Auto-Session Save:** Logs in once and saves the session permanently.
* ⚡ **Fast Execution:** Instantly reads and replies to commands.
